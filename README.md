## WEB KEPENDUDUKAN
<p><b>Aplikasi Kependudukan Montalat</b></p>

## Setup
- rename file .env.example dengan .env
- buat database (db_surat_disposisi) dan ganti di env

## Command
- composer install
- php artisan optimize:clear
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
